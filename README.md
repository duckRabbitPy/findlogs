# Findlogs

</br>

## A simple CLI tool for locating stray console.log statements

</br>

## How to install:

</br>

First run:

```shell
npm install findlogs
```

Then run:

```shell
npm link
```

or

```shell
npm link findlogs
```

</br>

### How to use:

```shell
cd myRootFolder
```

```shell
findlogs
```

</br>

#### Output:

```shell
❯ findlogs
🔎 Found  3 console.logs in index.js
```
